# Title	Standardized QA SDET Code examination 


###  Prerequisite
- Java 11
- Junit 5
- Jakson jars

###  Run Program
- Unzip folder 
- Go to Jar folder
- Open command prompt
- Set working path to jar folder
- Set Java Home and path if not
- Run Java -jar SDET.jar


###  Run Junit
- Unzip folder 
- Go to Jar folder
- Open command prompt
- Set working path to jar folder
- Set Java Home and path if not
- To run project use below command:
	java -cp SDET.jar  com.qa.sdet.test.TestRunner
